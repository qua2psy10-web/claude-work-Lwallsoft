# Lwallsoft — L型擁壁 設計計算アプリ

L型擁壁(逆L型・逆T型を含む片持ばり式擁壁)の安定計算と部材断面照査を行う、単一HTMLファイルのWebアプリです。claude-work-Gwallsoft で作成した重力式擁壁計算アプリのL型擁壁版として作成しました。

## 使い方

`index.html` をブラウザで開くだけで動作します(外部ライブラリ・通信は不要)。左側の入力欄を変更すると即時に再計算され、断面図・判定表が更新されます。「計算書を印刷」ボタンでそのまま計算書として印刷できます。

## 計算内容(道路土工 擁壁工指針 準拠)

計算はすべて奥行き 1m あたりで行います。

### 土圧算定 — 試行くさび法

- かかと版末端を通る鉛直な**仮想背面**(高さ H' = H + B3・tanβ)に作用する主働土圧を、すべり角 ω を走査する試行くさび法で算定します。壁面摩擦角は δ=β(地表面勾配)。
  - 常時: P(ω) = W・sin(ω−φ) / cos(ω−φ−δ)、載荷重 q はくさび重量に加算
  - 地震時: P(ω) = W・{sin(ω−φ) + k<sub>h</sub>・cos(ω−φ)} / cos(ω−φ−δ)(物部・岡部式と等価)、載荷重は無視
- 土圧合力の作用高さは H'/3、水平成分 Ph=P・cosδ、鉛直成分 Pv=P・sinδ。
- 地震時は躯体および仮想背面より前の背面土砂にも慣性力 k<sub>h</sub>・W を作用させます。

### 安定計算

つま先版先端まわりのモーメント集計から合力作用位置 d、偏心量 e を求めて照査します。

| 照査 | 常時 | 地震時 |
|---|---|---|
| 転倒(偏心量) | \|e\| ≦ B/6 | \|e\| ≦ B/3 |
| 滑動 Fs = (ΣV・μ + c<sub>B</sub>・B) / ΣH | ≧ 1.5 | ≧ 1.2 |
| 支持(地盤反力度 qmax) | ≦ qa(常時) | ≦ qa(地震時) |

地盤反力度は e ≦ B/6 で台形分布、e > B/6 で三角形分布(有効載荷幅 3d)とします。

### 部材断面照査 — 許容応力度法

単鉄筋長方形断面(b=1000mm、ヤング係数比 n=15)として σc・σs・τ を算出し、許容値と比較します。地震時は許容応力度を 1.5 倍に割増します。

- **たて壁**: 付け根断面。壁背面(高さ H−t3)に作用する土圧(壁面摩擦角 δ=(2/3)φ、変更可)+ 躯体慣性力
- **つま先版**: 付け根断面。地盤反力(上向き)− 底版自重(下向き)
- **かかと版**: 付け根断面。背面土重量+載荷重+底版自重(下向き)− 地盤反力(上向き)

## 主な仮定・注意事項

- 載荷重 q は安定計算では常時のみ考慮し、かかと版上にも載せています(V・Mr に算入)。
- つま先側の埋戻し土の重量および受働土圧は無視しています(安全側)。
- 転倒照査は擁壁工指針に合わせて偏心量 e により判定します(Mr/Mo の安全率判定ではありません)。
- 水圧・浮力は考慮していません(排水工が適切に設けられる前提)。
- 本アプリの結果は設計の検討補助を目的としたものです。実設計では技術者の照査・責任のもとでご使用ください。

## 検算

- β=0・δ=0 の常時土圧が Coulomb(Rankine)主働土圧係数 Ka=tan²(45°−φ/2) と一致すること、地震時土圧が物部・岡部式の閉形式解と一致することを自動テストで確認済みです。
