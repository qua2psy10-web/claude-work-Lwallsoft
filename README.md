# L型擁壁 設計計算ソフト（Lwallsoft）

道路土工 擁壁工指針に準拠した、L型擁壁（片持ばり式・逆L型を含む・直接基礎）の安定計算・部材照査と設計計算書（A4帳票／PDF）出力を行うブラウザアプリです。claude-work-Gwallsoft の重力式擁壁計算ソフトと同じ画面構成・帳票スタイルで作成しています。

## 使い方

ES モジュールを使用しているため、`file://` 直接ではなく HTTP 経由で開いてください。

```bash
python3 -m http.server 8000
# → http://localhost:8000/index.html
```

- 左側パネルで形状寸法・土質・材料・荷重・安定計算条件・配筋を入力すると、右側に計算書（A4ページ）が即時生成されます。
- 「印刷 / PDF保存」ボタン（またはブラウザの印刷機能）で PDF として保存できます。
- 「入力を保存(JSON)」「読込」で入力データの保存・再読込ができます。
- ツールバーのプリセットから、標準（H=3.0m レベル）・嵩上げ盛土・H=5.0m・逆L型・衝突荷重考慮・水位/揚圧力考慮・隅角部ハンチの7種を呼び出せます。

## 計算内容（道路土工 擁壁工指針 準拠）

計算はすべて奥行き 1m あたりで行います。

- **躯体自重**: 座標法（断面積・重心）。L型断面（テーパー付きたて壁＋底版）に対応。隅角部ハンチ（たて壁背面と底版天端の入隅を埋める三角形、高さ・幅を入力）を任意で付加でき、自重に算入・背面土砂を控除
- **主働土圧**: 試行くさび法（仮想背面）。仮想背面はかかと版末端を通る鉛直面（高さ H′ = H + min(B3/n, 嵩上げ高さ)）、壁面摩擦角 δ は仮想背面位置の地表面勾配（上限φ、レベル到達済みなら0）。常時（載荷重を加算）／地震時（慣性力 kh・W を考慮、θ=tan⁻¹kh の物部・岡部式と等価）。β=0・δ=0 でクーロン主働土圧係数と、地震時は物部・岡部式と、無限長斜面の極限では Rankine の理論解と一致することを検証済み
- **背面土砂形状**: レベルのほか、嵩上げ盛土（たて壁天端から勾配 1:n で嵩上げ高さまで立ち上がり、その先レベルの折れ線地表面）に対応。勾配は 1:n 形式で入力。嵩上げ高さに負値を入力すると天端からの**落差**（土砂面が天端より低いレベル地形、土圧作用高 H′ = H − 落差）として扱う
- **背面土砂**: かかと版上の矩形部＋嵩上げ盛土部（勾配三角形＋水平部矩形）を鉛直荷重として算入
- **躯体・背面土砂の慣性力**: H = kh・W
- **作用力の集計**: 偏心量 e = B/2 −(ΣV・x − ΣH・y)/ΣV、M = ΣV・e（つま先版前面まわり）
- **地盤反力度**: 台形分布／三角形分布の自動判定
- **水位・揚圧力（オプション）**: 前面・背面水位（底版下面から）を入力。背面水位以下の土圧算定は水中単位体積重量 γ′、仮想背面に背面水圧（前方へ押す）・前面に前面水圧（抵抗側）の静水圧 PW=1/2・γw・Hw² を作用。揚圧力は底版下面の台形分布 UP=(uP1+uP2)/2・B（オン/オフ可）。水位考慮時の常時ケースは「浮力考慮」となり、地震時・衝突時は慣行に従い浮力無視で算定。部材照査ではたて壁に背面水圧を加算し、つま先版・かかと版には揚圧力分布を上向き荷重として算入
- **衝突荷重（オプション）**: 荷重強度 P (kN/m)・作用高さ h（底版下面から）を入力し、土圧と同方向の水平力 H = P・L として「衝突時」ケースで考慮（慣行に従い常時土圧・活荷重なしで算定）。安定照査の許容値は衝突時用を別設定、部材照査の割増係数は地震時と同値（既定1.5）。作用高さが底版より上（h > t3）の場合はたて壁の断面力にも算入
- **安定照査**: 転倒（|e|≦B/n）・滑動（Hu/H≧Fs、水平力0以下は算定不能表記）・支持（qmax≦qa）。常時／地震時／衝突時で許容値を別設定
- **部材計算**: たて壁・つま先版・かかと版の付け根断面を RC 単鉄筋長方形断面（b=1000mm、n=15）として許容応力度法で照査。σc（曲げ圧縮）・σs（鉄筋引張）・τ（せん断）を算出し、地震時は許容応力度を割増（既定1.5倍）。たて壁は壁背面土圧（δ=2/3・φ）＋躯体慣性力、つま先版・かかと版は地盤反力と上載荷重の差引きで断面力を算定

## 帳票構成

- 目次（ページ番号付き）
- 第1章 設計条件（断面形状図・全体形状図・土砂条件・計算条件・荷重条件・安定計算条件・部材計算条件・荷重の組み合わせ）
- 第2章 計算結果一覧（作用力・転倒/滑動/支持照査・部材照査の判定一覧）
- 第3章 作用力の算定（座標法・試行くさび法の公式、ケース毎の土圧計算とω-PA曲線グラフ、作用力集計、地盤反力度と反力分布図）
- 第4章 安定計算（ケース毎の転倒・滑動・支持照査）
- 第5章 部材計算（たて壁設計用土圧の算定〔くさび図・ω-PA曲線〕・断面力の内訳・配筋断面図・応力度照査）

## 検証

計算エンジンは理論解・手計算と照合済みです。

```bash
node tests/verify.mjs
# → 結果: 86 件一致 / 0 件不一致
```

- 常時土圧が β=0・δ=0 でクーロン主働土圧係数 Ka=tan²(45°−φ/2)、地震時土圧が物部・岡部式の閉形式解と一致
- すべり角 ω=45°+φ/2、上載荷重の寄与 Ka・q・H を確認
- 偏心量・滑動安全率・地盤反力度・部材応力度を手計算と照合
- 逆L型（つま先版なし）・盛土勾配・全プリセットで NaN・例外が発生しないことを確認

## 適用範囲・注意

- L型擁壁（片持ばり式、テーパー付きたて壁・前面つま先版・背面かかと版、逆L型を含む）、盛土部擁壁、直接基礎を対象としています。背面土砂はレベルまたは嵩上げ盛土（勾配 1:n ＋ 天端レベル）です。
- 仮想背面法により土圧を算定し、仮想背面より前の背面土砂は鉛直荷重として扱います。土圧合力の作用高さは H′/3 としています。
- つま先側の埋戻し土重量・受働土圧は考慮していません（安全側）。水位・揚圧力はオプションで考慮できます（背面土砂の鉛直重量は水位以下でも湿潤重量のまま、浮力は底版下面の揚圧力で考慮する簡便法）。
- 実務での使用にあたっては、適用条件・計算結果を必ず技術者が確認してください。
